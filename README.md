# GitHub Pages Deployment Workflow

Este projeto publica um site estático no GitHub Pages usando GitHub Actions.

## Conteúdo
- `index.html`: página principal do site (currículo online)
- `.github/workflows/deploy.yml`: workflow de deploy
- `README.md`: documentação do projeto

## Como funciona
Sempre que houver um push na branch `main` alterando o arquivo `index.html`, o GitHub Actions executa o workflow e publica o site no GitHub Pages.

## Sobre a página
A página é um currículo online moderno, responsivo e com suporte a impressão (“Imprimir / PDF”), com as principais informações profissionais.
